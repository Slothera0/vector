## Vector

A simple Vector library in C.

There a very few function actually, i will created more if i need it.

# vec_append:
Add a new element on the vector, if not enough place to do it realloc more place

# vec_copy:
Return a copy of a vector

# vec_create:
Create a new vector with the given size

# vec_free:
Free a vector

# vec_iter:
Iter on the vector and use the given function on it
